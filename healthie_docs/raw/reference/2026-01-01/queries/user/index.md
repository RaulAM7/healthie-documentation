Fetch a user by ID

## Arguments

[`clear_notifs` ](#argument-clear-notifs)· [`Boolean` ](/reference/2026-01-01/scalars/boolean)· Whether to clear notifications for the fetched user.

[`email` ](#argument-email)· [`String` ](/reference/2026-01-01/scalars/string)· DEPRECATED: Does nothing.

[`id` ](#argument-id)· [`ID` ](/reference/2026-01-01/scalars/id)· The ID of the user to fetch.

[`or_current_user` ](#argument-or-current-user)· [`Boolean` ](/reference/2026-01-01/scalars/boolean)· If true and the specified user is inaccessible, returns the current user instead.

## Returns

[`User`](/reference/2026-01-01/objects/user)
